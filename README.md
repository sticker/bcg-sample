# bcg-sample

## install

```
npm install
```

## make .env

```
echo 'ADMIN_WALLET_MNEMONIC="this is sample mnemonic phrase"' > .env
```

## contract build

```
npm run build
```

## contract migrate to testnet

```
npx truffle migrate --network testnet
```

## frontend install

```
cd frontend
npm install
```


## frontend start

```
npm run start
```
